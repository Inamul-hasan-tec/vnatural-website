# VNatural Brand Color Reference

## Logo Colors Extracted
- **Dark Forest Green** (Primary): #006B3F
- **Lime Green** (Secondary): #7ED321 / #8BC34A
- **Orange** (Accent): #FF6B35 / #FF7043

## Color Mapping for Updates

### Old Colors → New Colors
- `from-green-500 to-emerald-600` → `from-vn-forest to-vn-lime`
- `from-green-600 to-emerald-600` → `from-vn-forest to-vn-lime`
- `from-emerald-500 to-green-600` → `from-vn-forest to-vn-lime`
- `from-emerald-600 to-green-600` → `from-vn-forest to-vn-lime`
- `text-green-600` → `text-vn-forest`
- `text-emerald-600` → `text-vn-forest`
- `bg-emerald-` → `bg-vn-forest` or `bg-vn-lime` (depending on context)
- `border-emerald-` → `border-vn-forest` or `border-vn-lime`

### Accent Colors (Keep Diverse but Adjust)
- Blue gradients: Keep for commerce/tech sections
- Purple gradients: Keep for medical/AI sections  
- Orange gradients: Update to use `vn-orange` for consistency
- Red gradients: Can use `vn-orange` or keep for specific emphasis
